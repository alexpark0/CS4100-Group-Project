### CS4100 Group Project SP26

## Movie recommendation model using CNN and LLM to generate plot descriptions.

With so many movies out there and the length of a movie being such a large time commitment, 
it is difficult to find a new movie that you might like. It’s also sometimes a struggle to 
find a movie that has the same ‘vibe’ as another movie, and a viewer might not know what draws 
them to that specific ‘vibe.’ Streaming services already recommend shows and movies based on 
user preferences, but some people remain unsatisfied with those recommendations and stay unsure 
of what they want to watch next. A good movie recommendation model with customizable inputs can solve this problem.
